# SITE FICTICIO DE DENTISTA COM FIM EDUCACIONAL.

## PROJETO FICTICIO SEM FIM COMERCIAL.

## PORQUE ESTE PROJRTO?
Este projeto ficticio tem como objetivo em colocar em pratica os aprendizados e habilidades adquiridos durante longa jornada de estudo. É um projeto que relata muita quebra-cabeça, muitas pesquisas e conhecimentos colocados em prática. Vale ressaltar que é um projeto ficticio com aspecto do mundo real.

## TECNOLOGIAS E FERRAMENTAS UTILIZADAS.
Este projeto contem o uso de HTML, CSS E JAVASCRIPT.
A estrutura dos textos e imagens utilizados é feita pelo HTML. Todoas as parte de apresentações como, estilos visuais, botões dos menus são feitas pelo CSS. E na página "SOBRE NÓS", a interatividade automatica das imagens (SLIDE SHOW) é controlada pelo JAVASCRIPT.

## FONTES DAS IMAGENS UTILIZADAS.
Neste projetos, as imagens utilizadas são de uso gratuito vindo de fontes que as disponibilizam de forma gratuita. Portanto quase 100% das imagens utilizadas é da site FreePik.

## QUEM PODE VER ESTE PROJETO?
Este projeto é publico e pode ser visto pelos colaboradoras que queriam oferecer feedbacks produtivos de melhorias, usar o projeto de inspiração para complementar o desenvolvimento de seus aprendizados, também pelas Empresas ou Pessoas Recrutadoras que queriam oferecer oportunidades de empregos.
